# Password Generator

Using tkinter, this python application allows one to save, create, and update passwords for user logins

## Installation

Install (2) files and run .exe
  PasswordManager.exe 
  logo.png

## Usage

Search: allows you to find your login password for the site
without revealing user email

Password: Custom password input can be entered, or user may click random password generator button

Add: Saves website, user login and password information. Will also overwrite and update login and password information if already exists in JSON data.
